# Installation

1. Download the latest release
2. Move **WindowsGSM.Foundry.cs** folder to plugins folder
3. Update parameters in file Foundry.cs (Password, ports etc...)
4. Click **[RELOAD PLUGINS]** button or restart WindowsGSM
